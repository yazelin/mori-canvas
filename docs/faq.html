常見問題

試玩、錄音、房間資料、自架 —— 最常被問的問題,老實回答。沒答到的,歡迎到 GitHub 開 issue 問。

試玩與帳號

點開很慢,甚至一片白?

正常。試玩站跑在 Render 免費方案,閒置 15 分鐘會休眠 —— 第一個點開的人要等約 30–60 秒冷啟動,白畫面多等一下就好,醒來之後就順了。要穩定常開,自架或換 Render 付費方案。

要註冊嗎?要錢嗎?

不用註冊,進場輸入一個名字就能用。試玩站免費,AI 走站長的 key,有 per-IP 限流;想用自己的額度,⚙ 設定可填「用你自己的 AI」。自架也是零授權成本(全 MIT),AI 用自己的 Groq key —— Groq 有免費額度,到 console.groq.com/keys 申請即可。

進去不知道從哪開始?右上「範例」按鈕(或空白板上的提示)可一鍵載入 5 個範例板;初次進入也有 6 步互動導覽,之後想重看按「?」。

手機能用嗎?

可以。手機能看、能編、能錄音(雙指縮放、單點卡片開選單),還能裝成 PWA:Android / Chrome 在選單按「安裝 App」,iPhone / Safari 用「分享 → 加入主畫面」,之後就是全螢幕、像原生 App。

錄音與 AI

麥克風沒反應?

依序檢查三件事:① 瀏覽器有沒有允許麥克風(網址列的鎖頭圖示);② 系統的輸入裝置選對了沒,有些筆電預設選到沒在收音的那個;③ 錄音只在 https 或 localhost 下可用 —— 這是瀏覽器規定,http://區網IP 會被直接擋。自架要讓手機錄音,用 npm run setup + npm run start:lan 跑 HTTPS 版。

講中文以外的語言可以嗎?

語音辨識用 Whisper,本身是多語模型,講英文等其他語言也認得。不過介面與 AI 產出的卡片目前以繁體中文為主,中文輸出會由程式硬轉成繁體(台灣用語),不靠模型自律。

AI 把話整理錯了怎麼辦?

先說它怎麼降低錯誤:pipeline 是兩段式 —— 先清稿(收斂重複字詞、刪「嗯、那個」、重斷句)再畫卡,口語贅字不會被抄上板。還是錯的話,三條路:用講的改(把官網那張改寫成 RWD 重構、把 3 號改成風險)、雙擊卡片直接改字、或 Ctrl+Z 復原。

房間與資料

房間誰能進?

知道房號或連結的人就能進,沒有另外的登入 —— 房號就是密碼,別公開貼。另外注意:試玩站的分享面板看得到「進行中房間清單」(含房號),所以試玩站上別放不想被外人看到的內容;要私密,自架。

資料存在哪?會保留多久?

自架:每個房間存在主機的 .data/<房號>.bin,重啟自動還原、沒有過期,資料不出你的機器。試玩站:存在 Render 容器內,重新部署或休眠重啟就清空 —— 別放敏感內容,想留就用「下載畫板檔(.json)」帶走。

可以匯出嗎?

可以,工具列「匯出」有五種:白板摘要(AI 依圖表類型整理成一頁)、白板紀錄 HTML(摘要 + 全程逐字稿,單檔離線可讀)、Markdown、PNG、畫板檔 .json(完整存檔,可匯入還原、傳給別人接著編)。

自架與其他

跟 Miro / FigJam 差在哪?

定位不同。它們是「人畫」的泛用協作白板,功能廣度比 Mori Canvas 大得多;Mori Canvas 的核心是語音 → AI 自動長板 —— 你開會講話,便利貼、連線、分區自己長出來。另外它可自架、全 MIT、零授權成本,資料留在自己機器。要泛用畫圖工具,用它們;要「會議講完板就好了」,用這個。

離線(純內網)能用嗎?

可以。server 跑在自己機器,STT 切「自訂 · 本機 whisper-server」,LLM 用本機 Ollama(或設 LLM_LOCAL_ONLY=1 強制全走本機),全程資料不出網,沒有對外網路也能開會。試玩站本身則需要網路。

怎麼回報問題、許願功能?

到 GitHub 開 issue。bug、想要的功能、甚至「跑出簡體字」這種小事都歡迎 —— 這專案還在積極開發,回報就是幫忙。
