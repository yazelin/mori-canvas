Mori Canvas 共筆白板

講話,或貼上逐字稿 —— AI 自動把重點整理成便利貼與關係圖,大家在同一張白板上即時協作。

會議語音 → 逐字稿 → AI agent → 便利貼+連線 → 多人 live 白板
立即試玩 先看示範板 看範例教學 GitHub
示範站冷啟動說明。試玩站跑在 Render 免費方案,閒置 15 分鐘會休眠 —— 第一個點開的人要等 30–60 秒,之後就正常了。本頁已經順手幫你預熱;等不及或想穩定使用,自架只要一顆 Rust binary。

30 秒看懂

  1. 講話。開著會正常講,停頓一下就自動斷句送出;不想講話,貼現成逐字稿也行。
  2. AI 整理成卡。兩段式:先把贅字、錯字、斷錯的句子清乾淨,再整理成按性質上色的便利貼與關係連線。
  3. 大家即時共筆。同事掃 QR 用瀏覽器進來,零安裝;拖卡、改字、連線,所有人即時同步。

看它跑起來

實際流程:貼上逐字稿讓 AI 長出白板,再把成果匯出成可閱讀、可分享的檔案。

三分鐘開始

  1. 開房。點「立即試玩」,輸入名字進場(會自動產一個新房號)。初次進入有引導加 6 步互動導覽,之後隨時可從「?」重開。
  2. 講話,或貼逐字稿。按左下「開始會議記錄」開講;或把現成逐字稿貼進面板,丟給 agent。
  3. 分享 QR。工具列「分享 / QR」,同事掃了就進來,零安裝即時同步。
  4. 匯出。散會按「匯出」:白板紀錄 HTML、Markdown、PNG,或下載可還原的畫板檔。
