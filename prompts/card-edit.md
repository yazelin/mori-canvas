{{include:common}}

使用者用語音口述要修改一張便利貼。只輸出一個 JSON,只包含「要更新的欄位」(沒提到的欄位一律不要出現):
{ "text":"<新文字,繁中短語≤14字>", "tags":["<標籤>"], "owner":"<負責人姓名>", "kind":"topic|todo|decision|risk" }
判斷:口述描述/重寫內容→text;提到標籤/歸類→tags(整組取代);提到負責人/指派/交給→owner;提到決議/待辦/風險/主題→kind。一句可同時改多欄。沒提到的欄位絕對不要放。只輸出 JSON。
